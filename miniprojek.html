Sejarah Penemuan HTML

HTML (Hypertext Markup Language) ditemukan oleh Tim Berners-Lee, seorang ilmuwan komputer asal Inggris, pada akhir 1980-an dan awal 1990-an. Berners-Lee menciptakan HTML sebagai bagian dari proyeknya untuk membuat World Wide Web (WWW) ketika bekerja di CERN (Organisasi Eropa untuk Riset Nuklir) di Swiss.

HTML digunakan untuk membuat dan mengatur halaman web, memungkinkan pengguna menghubungkan dokumen dengan tautan (hyperlinks). Tim Berners-Lee juga menciptakan browser web pertama dan server web pertama untuk mengimplementasikan konsep World Wide Web.

Penemuannya menjadi dasar dari perkembangan internet seperti yang kita kenal saat ini.
